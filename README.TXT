PROJECT TITLE: Perimeter Assignment Runner
PURPOSE OF PROJECT: To select coordinate of any shape(from Directory)  and return their :
1) Perimeter.
2) Number of Points.
3) Average Length.
4) Largest Length.
5) Largest X coordinate.
6) Compare many shape and return name of shape with largest perimeter.
7) Output largest perimeter from different input shapes coordinates.
